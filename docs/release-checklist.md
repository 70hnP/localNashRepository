# Local Production Readiness Checklist

## Pre-release
- [ ] `npm install` ejecutado correctamente.
- [ ] `npm run build` sin errores.
- [ ] Navegación entre tabs validada.
- [ ] Formularios con validaciones mínimas verificados.
- [ ] Persistencia local validada (guardar/recargar/reset).
- [ ] Scorecards recalculan ante cambios.
- [ ] Estados vacíos muestran mensajes claros.

## Release
- [ ] Versión etiquetada `v0.1.0-local-mvp`.
- [ ] README actualizado con instrucciones de ejecución.
- [ ] Seed de ejemplo cargado por defecto.

## Primera semana
- [ ] Verificar uso real de cada módulo.
- [ ] Revisar calidad de datos ingresados.
- [ ] Ajustar umbrales de scorecards según evidencia.
